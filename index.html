Intro to jQuery

What is jQuery?

jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript.

jQuery.com

Why use jQuery?

  1. Document Traversing

    jQuery streamlines the selection of DOM elements and attribute editing

  2. Event Handling

    jQuery allows you to bind code to specific user actions

  3. Animating

    jQuery takes the pain out of coding simple or complex animations

  4. Ajax

    jQuery wraps communication with the server and handles parsing of response data
